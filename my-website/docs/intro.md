# Introducción

Este sitio web contiene todo el conocimiento común en base a todo el feedback dado en las clases de tarde de ISPP, recopilado por los distintos grupos.

## Estructura

Se encuentra una sección para cada grupo de tarde (del grupo 6 al grupo 11), donde mostrarán su feedback específico, y también una sección general donde todos los grupos consolidarán el feedback global para todos los grupos. Igualmente, se apunta para cada presentación una checklist con los puntos mínimos que esta debe tener.
